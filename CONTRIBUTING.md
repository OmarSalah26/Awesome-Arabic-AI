# Contributing to Awesome Arabic AI

Thank you for your interest in contributing to the **Awesome Arabic AI** project! We aim to maintain a professional, high-quality, and authoritative hub for the community.

## üìú Contribution Guidelines

To ensure consistency and quality, please follow these rules when submitting a pull request:

### 1. Relevance
All submissions must be directly relevant to Arabic Large Language Models (LLMs), Speech-to-Text (STT), Text-to-Speech (TTS), or related linguistic tools/data.

### 2. Format & Style
- **Standardized Metadata**: Every entry must include:
  - Model/Project Name
  - Developer/Organization
  - Parameter Size (if applicable)
  - License Type
  - Direct Link (GitHub or Hugging Face)

### 3. Submission Protocol
1. **Fork** the repository and create your branch from `main`.
2. **Add** your resource to the relevant section or directory.
3. **Verify** all links are active and point to the correct source.
4. **Pull Request Description**: Provide a clear explanation of why this resource is an advancement for the Arabic domain.
5. **Wait** for review by the architectural maintainers.

If you are contributing large amounts of data or new categories, please use the thematic directories:

- `speech/`: ASR, TTS, and Diacritization resources.
- `data/`: Datasets, Corpora, and Benchmarking metadata.
- `tools/`: Preprocessing and evaluation scripts.
